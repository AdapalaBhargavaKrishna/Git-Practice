# 🚀 Git Practice

A beginner-friendly repository created to practice **Git** and **GitHub** commands while learning version control.

---

## 📌 About

This repository is used to understand the complete Git workflow, including:

* Initializing a Git repository
* Tracking files
* Creating commits
* Using Conventional Commit messages
* Working with branches
* Connecting to GitHub
* Pushing changes to a remote repository

## 📂 Project

### 🧮 Calculator

### Features

* ➕ Addition
* ➖ Subtraction

## 📚 Git Commands Practiced

```bash
git init
git status
git add .
git commit -m "feat: add basic calculator functions"
git branch -M main
git remote add origin <repository-url>
git push -u origin main
git push
git config --list
git config --global user.name
git config --global user.email
```

---

## 🌿 Conventional Commits

Examples used in this repository:

```text
feat: add basic calculator functions
fix: resolve division by zero
docs: update README
refactor: improve calculator structure
```
